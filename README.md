# Trader-Sentiment-Analysis

Objective
Analyze relationship between Bitcoin market sentiment and trader behavior.

Methodology
1. Data cleaning
2. Timestamp conversion
3. Dataset merging
4. Feature engineering
5. Sentiment analysis

Key Insights
1. Traders earn higher profits during Greed sentiment.
2. Fear sentiment increases trading activity.
3. Position sizes increase during bullish market sentiment.

Strategy Recommendations
1. Reduce leverage during Fear sentiment.
2. Increase position size for high-performing traders during Greed sentiment.
